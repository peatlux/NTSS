HOW TO install geophone listener on new machine:
- have geophone in same network
- on laptop, browse to rs.local
- in Settings > Datacast, add address of laptop and port 8888

- in Terminal, install python-osc with:
pip install python-osc ## only  needed once

- in Terminal, run this to start listen/forwarding:
~/Seafile/BiophonySetup/code/geo/geoStart.sh
